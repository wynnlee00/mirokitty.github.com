Guide

Guide

01 AWS Account 페이지에 접속합니다.

02 상단에 ‘계정 ID’ 정보를 확인하여 입력합니다. (숫자 12자)

Guide

01 AWS Console에 로그인 후 IAM Dashboard 페이지에 접속합니다.

02 좌측 메뉴에서 'Policies'를 선택한 후 'Create Policy' 버튼을 클릭합니다.

03 'Create Your Own Policy' 클릭 후 Policy Name, Policy Document (팝업 내용 복사)를 입력하고 'Create Policy' 버튼을 클릭합니다.

04 좌측 메뉴에서 'Groups'를 선택한 후 'Create New Group' 버튼을 클릭합니다.

05 그룹 이름을 입력하고 위에서 추가한 Policy를 첨부한 후 'Create Group' 버튼을 클릭합니다.

06 좌측 메뉴에서 'Users'를 선택한 후 'Add User' 버튼을 클릭합니다.

07 Access type은 Programmatic access를 선택하고 User를 추가합니다.

08 'Add User' 마지막 단계에서 Access Key ID, Secret Access Key 정보를 각각 복사하여 위에 입력합니다.
