Account

Account list

비용 조회를 위해서는 사용 중인 AWS 계정 연동이 필요합니다.
(*계정 정보가 정상적으로 추가되었다면 해당 데이터를 불러오는데 약간의 시간이 소요될 수 있습니다.)

- AWS 계정 등록 : 솔루션 관리자만 가능
- AWS 계정 조회/수정/삭제 : 솔루션 관리자, 회사 Default 그룹에 속한 사용자 가능

Edit 화면
  1. 수정/조회를 원할 경우 해당 계정의 ‘Edit’ 버튼을 클릭합니다.
  2. 수정이 필요한 항목을 수정한 후 ‘Save’ 버튼을 클릭합니다.
  3. Account List 화면에서 ‘Bucket’ 버튼이 활성화 된 상태로 보인다면 S3 Bucket 정보를 추가로 입력해야 합니다.
    (Linked Account를 제외한 나머지 타입의 계정은 데이터를 불러오기 위해 Bucket 정보가 필요합니다.)
  4. ‘Bucket’ 팝업 창에서 Region, Bucket Name을 입력 후 ‘Save’ 버튼을 클릭합니다.
Delete 화면
  1. 삭제하고자 하는 AWS 계정의 ‘Delete’ 버튼을 클릭합니다.
  2. ‘Delete’ 팝업 내용을 확인 후 ‘OK’ 버튼을 클릭합니다.
    (*AWS 계정 정보를 삭제할 경우 솔루션에서 해당 계정의 관련 정보가 모두 삭제됩니다.)

User List

사용자 초대 화면

Default 그룹에 속한 사용자는 회사의 다른 사용자를 초대할 수 있습니다.

  1. ‘Create User’ 버튼을 클릭합니다.
  2. 사용자 초대 팝업 창에서 필수 입력 값인 Email ID를 입력, Group을 선택한 후 ‘OK’ 버튼을 클릭합니다.
  3. 초대 메일 발송 완료 팝업 창을 확인 후 ‘OK’ 버튼을 클릭합니다.
  4. 초대 메일을 받은 사용자는 해당 메일에서 가입 후 이용을 시작합니다.
초대 메일 재발송
  1. 초대 메일의 유효 시간은 72시간이며, 만료될 경우 활성화되는 ‘Send Email’ 버튼을 클릭합니다.
  2. ‘Send Email’ 팝업 내용을 확인 후 ‘OK’ 버튼을 클릭합니다.
사용자 탈퇴/삭제
  1. 탈퇴 또는 삭제하고자 하는 이메일의 ‘Withdraw‘ 또는 ‘Delete’ 버튼을 클릭합니다.
    - Withdraw : 가입한 사용자의 경우
    - Delete : 초대했으나 가입하지 않은 사용자의 경우
  2. ‘Withdraw User’ 또는 ‘Delete User’ 팝업 내용을 확인 후 ‘OK’ 버튼을 클릭합니다.
비밀번호 초기화 메일 발송
  1. 비밀번호 초기화 메일을 발송할 이메일의 ‘Password Reset’ 버튼을 클릭합니다.
  2. ‘Reset Password’ 팝업 내용을 확인 후 ‘OK’ 버튼을 클릭합니다.

Group List

한 회사에서 다수의 AWS 계정을 사용 중이고 관련 담당자가 많을 때 유용한 그룹 관리 기능을 제공합니다.
Default 그룹에 속한 사용자는 그룹 관리 기능을 사용할 수 있습니다.

Create 화면
  1. 그룹 생성을 위해 ‘Create Group’ 버튼을 클릭합니다.
  2. 그룹명을 입력하고 해당 그룹에 속할 AWS Account, User를 선택 후 ‘Create’ 버튼을 클릭합니다.
Edit 화면
  1. 그룹 수정을 위해 해당 그룹의 ‘Edit’ 버튼을 클릭합니다.
  2. 수정이 필요한 항목을 수정한 후 ‘Update’ 버튼을 클릭합니다.
Delete 화면
  1. 그룹 삭제를 위해 해당 그룹의 ‘Delete’ 버튼을 클릭합니다.
  2. ‘Delete’ 팝업 창을 확인 후 ‘OK’를 클릭합니다.

Change Password

비밀번호 변경을 원할 경우 ‘Change Password’ 메뉴를 클릭합니다.
현재 비밀번호, 새 비밀번호, 새 비밀번호 확인을 입력 후 ‘Save’ 버튼을 클릭합니다.

‘비밀번호 변경 완료’ 팝업 내용을 확인 후 ‘OK’ 버튼을 클릭합니다.
(*클릭 후 로그인 페이지로 이동합니다.)
