About Virtual Test Lab

Virtual Test Lab 은 여러분이 준비 중인 스마트 홈 서비스에서 연동하는
LG 스마트 제품들의 원활한 테스트를 위해, 실제 제품의 주요 기능을 갖춘 가상 제품과
이를 제어할 수 있는 에뮬레이터를 제공하는 서비스입니다.

Virtual Test Lab 바로 가기

Virtual Test Lab의 특장점

가상 제품의 활용

실제 제품을 준비하는데 필요한
물리적인 공간이나 비용의 제약 없이,
가상 제품을 생성함으로써 실제 제품과
동일한 수준까지 테스트가 가능합니다.

동일한 사용 경험 제공

생성된 가상 제품은 실제제품과 함께
제품 리스트에서 조회, 삭제가 가능하며,
제품과 유사한 형태의 에뮬레이터 UI로
일관된 사용 경험을 할 수 있습니다.

직관적인 테스트 방식 제공

제품과 스마트 홈 서비스간의
인터페이스가 정상적으로 이루어지는지,
결함은 없는지, 직접 눈으로 확인하면서
개선해 나갈 수 있습니다.

Virtual Test Lab의 구성

  • 1가상 제품

    '가상 제품'은 스마트 홈 서비스를 지원하는 LG 스마트 제품을
    LG SmartHome Platform 상에 "가상 (Virtual)" 으로 구축한 제품을 의미합니다.
    실제 제품의 주요 기능을 동일하게 제공하며, API 를 통해 손쉽게 조회, 제어 등을 할 수 있습니다.

  • 2에뮬레이터

    가상 제품의 Display 와 Control Panel 역할을 하며,
    제품을 원격 또는 직접 제어할 수 있습니다.

    • 1 Display : 전원 On/Off, 설정된 모드 등 제품의 현재 상태를 시각적으로 확인할 수 있습니다.
    • 2 API 제공 기능 : LG SmartHome Platform의 API를 이용하여 원격으로 제어할 수 있는 기능들을 테스트할 수 있습니다.
    • 3 푸시 알림 : 에러 알림, 완료 알림 등 제품에서 제공하는 푸시 알림을 간편하게 테스트해 볼 수 있습니다.
    • 4 제품 설정 : API에서 지원하지 않는 기능이지만 제품의 현재 상태를 변경해볼 수 있도록 별도의 제품 설정 기능을 제공합니다.
    • 5 로그 : 가상 제품에서 발생한 이벤트의 결과 및 전체 로그를 확인할 수 있습니다.

Virtual Test Lab 이용가이드

step 1LG SmartHome
Developers 로그인
step 2서비스 생성
step 3사용자 등록 및
가상제품 생성
step 4에뮬레이터 실행
step 5테스트

  • 1LG SmartHome Developers 로그인

    가상 제품은 LG SmartHome Developers 에서 로그인 후 생성이 가능합니다.

  • 2Service 생성

    로그인 후 먼저 Service 를 생성해 주세요.
    각 Service 별로 연동 가능한 제품에 종류에 따라 가상 제품을 생성할 수 있습니다.
    서비스 생성 하러 가기

  • 3가상 제품 생성

    생성된 Service 와 연동 가능한 제품, 제품 모델을 선택하면 바로 가상 제품이 생성됩니다.
    가상 제품 생성 하러 가기

  • 4에뮬레이터 실행

    가상 제품이 생성되면, 가상 제품 별로 에뮬레이터를 실행할 수 있으며 가상 제품의 주요 원격 제어 기능들을 테스트해 볼 수 있습니다.
    가상 제품 목록 보러 가기

    • 에뮬레이터는 IE 10버전 이상, 크롬 31버전 이상에서 최적화되어 동작합니다. 브라우저가 최신 버전인지 확인해 주세요.
    • 실제 제품은 에뮬레이터로 테스트가 불가능합니다.

  • 5테스트

    에뮬레이터에서 제어한 결과를 준비중인 서비스에서 확인하거나, 반대로 서비스에서 원격 제어한 결과를 에뮬레이터에서 확인하면서
    원격제어 및 상태 조회 기능이 정상적으로 동작하는지 확인할 수 있습니다.

    • 1 가상 제품을 이용하여 개발중인 서비스를 테스트를 하기 위해서는 먼저 Virtual Test Lab에서 가상 제품을 생성해야 합니다.
    • 2 개발중인 서비스에서 가상제품이 등록되어 있는 개발자 계정 정보를 입력하면 가상제품 정보를 가져올 수 있습니다.
      • Search for Users (/v1/users) API를 사용하여 제품을 조회할 수 있습니다.
      • 조회된 제품 리스트 중 ‘"virtualDevice": 1’인 제품이 가상 제품입니다.
    • 3 개발중인 서비스에 등록된 가상제품과 동일한 닉네임의 에뮬레이터를 실행하면 가상제품의 상태가 확인하면서 테스트할 수 있습니다.
