LG SmartHome Developers는
LG SmartHome Platform을 활용하여
LG 스마트 제품 연동에 필요한 모든 API 및
이를 테스트할 수 있는 환경을 제공합니다.

LG SmartHome Developers APIs

  • Message API

    메시지 수/발신 관련 API 모음Messages API

    메시지는 대상 (사용자 또는 제품)에 따라 수신∙발신 할 수 있습니다.

    • 제품 또는 사용자에게 메시지 전송 요청
    • 제품 또는 사용자로부터 메시지 수신

  • Users API

    사용자 관련 API 모음Users API

    새로운 사용자를 등록하거나 등록된 사용자에 대한 수정 및 삭제를 할 수 있습니다.

    • 사용자 조회 및 등록
    • 등록된 사용자에 대한 정보 수정 및 삭제 요청

  • Device API

    제품 관련 API 모음Device API

    • 제품 등록
    • 등록된 제품에 대한 정보 조회/제어 명령
    • 제품 수정 및 삭제

  • Object API

    데이터를 입/출력 할 수 있는 Object Store API 모음Object API

    • 제품 상태 변경 알림 조회/해제/추가 요청
    • 펌웨어 검색/업로드/삭제 요청
    • 로그 조회/추가 요청 및 썸네일 리스트에 대한 조회/추가/삭제

  • Dialogue API

    자연어 처리를 위한 API 모음Dialogue API

    사용자 정보 및 입력 발화를 받아서 제품 제어 결과를 MQTT 채널로 응답합니다.

    • 새로운 자연어 처리 요청
    • 자연어 처리 요청에 대한 응답 수신

  • Misc. API

    부가 기능에 대한 API 모음Misc. API

    OTP 토큰 발급과 Critical 자원을 보호하기 위한 Lock에 대한 API를 포함합니다.

    • 새로운 Lock 및 Lock 해지 요청
    • 새로운 OTP 생성 및 OTP 조회 요청
    • 서버 연동을 위한 주소 조회
    • 세션에 대한 유효성 조회 및 발근된 세션에 대한 갱신 요청
