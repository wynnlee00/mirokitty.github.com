개발 환경에 따라서 제공하는 Max 수치의 값이 다릅니다.

개발환경 선택 시 주의 사항
• 개발이 완료된 내용으로 실제 서비스를 런칭하기 위해서는 OP 환경을 선택해서 관리자 승인 절차를 진행해야 합니다.
• 관리자 승인 절차는 승인요청 > 관리자 검증 > 승인완료 의 순서로 진행합니다.
• 각 단계별로 승인 진행에 대한 안내 메일을 발송하며, 단계별 상태는 ‘My Service’ 목록에서 확인 가능합니다.
• 승인절차 진행중에는 Service 수정 및 삭제는 불가능 합니다.

API 접근 권한

API 접근 권한을 설정해 주세요.

Read/Write : 제품의 현재 상태 정보를 조회하고, 설정을 변경할 수 있습니다.
Read : 제품의 현재 상태 조회만 할 수 있습니다.
