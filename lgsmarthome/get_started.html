Get Started with
LG SmartHome Developers

API, SDK를 통한 서비스 개발 및 Virtual Test를 위해
필요한 아래 6가지 가이드로 개발을 시작해 보세요.

  • 1

    Check out the Supported Products

    현재 준비중인 서비스(어플리케이션)에 연동할
    LG 스마트 제품의 종류, 모델, 기능을 먼저 확인해 주세요.


    • 제휴 내용에 따라 지원되는 제품 종류는 다를 수 있습니다.
    • 제품의 안전을 위해 일부 기능은 원격 제어가 제한됩니다.

    Supported Products 보기
  • 2

    Create Service ID

    개발을 진행하기 위해서는 Service 생성이 반드시 필요합니다.

    Service 생성 시, Service ID가 부여되고
    이는 지원제품, 대상국가, 개발환경 등을 규정하는
    Key 값으로 사용됩니다.


    • Service ID는 최대 20개까지 생성이 가능합니다.
    • 계정 등급(Admin/Developer)에 따라서 Service 생성이 제한될 수 있습니다.

    Service ID 생성하기
  • 3

    Connect User to Service ID

    생성한 Service ID에 LG 스마트 제품을 연동하려면
    그 제품을 등록한 User 정보가 필요합니다.

    필요한 User 정보 (Session ID, Session Key)는
    User API에서 생성할 수 있습니다.


    Supported Products 보기
  • 4

    Develop by API/SDK

    • LG 스마트 제품의 연동, 연동된 제품의 상태조회 및 제어 등등은
      각각의 API/SDK를 이용합니다.

      제품에서 사용 할 수 있는 Function Set 은
      해당 제품의 Features 에서 확인 후 입력해 주세요.


      Supported Features 확인하기

    • 각 API 별로 정의된 Parameter 값과 예제를 확인 후,
      Try (Test) 를 해 보면서 프로그래밍의 정합 여부를 확인해 보세요.


      • API는 Service ID 별로 등록된 User 및 Product 단위로
        세부 테스트가 가능합니다.

      API Reference 확인하기

    • SDK는 플랫폼 (iOS, Android, HTML) 별로 준비되어 있습니다.
      적합한 SDK를 다운로드 후 진행해 주세요.


      SDK 다운로드하기
  • 5

    Test by Virtual Product

    Virtual Test Lab에서는
    실제 제품의 주요 기능이 동일하게 동작하는 "가상 제품"과
    이를 제어할 수 있는 'Emulator'를 제공합니다.

    API 및 SDK를 이용해 개발이 완료되면,
    Virtual Test Lab에서 제품이 제대로 동작하는지 확인해 보세요.


    Virtual Test Lab 가기
  • 6

    Submit for Review

    서비스(어플리케이션) 런칭에 필요한 준비가 마무리되면,
    Service를 OP환경으로 새로 생성한 후 관리자에게 승인 요청을 해 주세요.

    승인이 완료되면 서비스(어플리케이션)를 바로 런칭 할 수 있습니다.


    승인 진행 단계
    승인 요청 관리자 검증 승인 완료 서비스 런칭
    각 단계별로 승인 진행 현황에 대한 안내 메일을 발송하며,
    현재 단계에 대한 상태는 'My Service' 목록에서 확인이 가능합니다.
  • 7

    Launch

    최종 승인이 완료되면, 서비스(어플리케이션)를 런칭할 수 있습니다.

    단, 고객이 제품을 연결해서 서비스(어플리케이션)를 이용하려면,
    미리 LG Account에 제품 등록이 되어 있어야 합니다.

    아래의 단계를 고객에게 가이드 해주시고,
    필요한 화면은 런칭 한 서비스(어플리케이션)에 함께 제공해 주세요.

1. 고객에게 미리 알려주세요!

LG Account에 제품 등록하기

LG 스마트 제품은 구매 후 LG Account에 제품을 등록해야 원격 제어가 가능합니다.
제품의 모바일 앱을 통해 LG Account 를 가입한 후 제품을 등록해 주세요.

2. 서비스 (어플리케이션)에 적용해 주세요!

Step 1
LG Account 로그인하기

제품을 등록한 LG Account 로 로그인 할 수 있도록 ID/PW 입력창을 제공해 주세요.


  • ID/PW 입력창 API는 SDK 문서를 통해 확인이 가능합니다.

SDK 다운로드 하기
Step 2
정보 제공 동의 받기

로그인이 완료되면, LG 스마트 제품을 통해 추가로 수급/제공되는 정보에 대한 별도 '동의' 절차를 진행합니다.

Step 3
연결 가능한 제품 목록 보여주기

동의까지 마무리가 되면, 해당 LG Account에 등록된 제품 목록을 불러올 수 있습니다.
이 목록에서 고객은 서비스 (어플리케이션)에 연결하고 싶은 제품을 선택해서 이용할 수 있습니다.


  • 연결 가능한 제품 목록 API는 API Reference에서 확인이 가능합니다.

API Reference 확인하기

추가 궁금하신 사항은 관리자에게 문의해 주세요.
